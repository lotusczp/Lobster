Lobster Script API


include(fileName)

Calling this API will load the specified file into the context. Similiar to "include" in c/c++.

Note: The file included should be in script folder.


lbMain.startTrans()

Calling this API will start all the channels. It's equivalent to clicking the "Start" button.


lbMain.stopTrans()

Calling this API will stop all the channels. It's equivalent to clicking the "Stop" button.


lbPoint.subscribe(dataName)

Calling this API will subscribe this data into the watch window. It's equivalent to select this point in the select data widget.


lbPoint.unsubscribe(dataName)

Calling this API will unsubscribe this data. It's equivalent to unselect this point in the select data widget.


value lbPoint.getValue(dataName)

Get the current value of identified by "dataName". The return value is a javascript object.


lbPoint.setValue(dataName, dataValue)

Set the point identified by "dataName" with the value "dataValue"


lbPoint.setCurveEnable(dataName, isEnable)

Set the point identified by "dataName" visible/invisible in the oscilloscope.


lbPoint.setCurveColor(dataName, colorString)

Set color of the curve identified by "dataName" in the oscilloscope. The colorString is defined by RGB value. For example, "#FF0000" refers to the red color.


lbPoint.setCurveGain(dataName, gain)

Set gain of the curve identified by "dataName" in the oscilloscope.


lbPoint.setCurveOffset(dataName, offset)

Set offset of the curve identified by "dataName" in the oscilloscope.


lbPoint.pullValue(dataName)

Pull the register value identified by "dataName" in the watchwindow.


lbPoint.setLogEnable(dataName, isEnable)

Set the point identified by "dataName" enable/disable in the Log system.


lbSystem.wait(milliSeconds)

The evaluation of script will stop for some milliSeconds identified by parameter.

Note: This "wait" will not occupy cpu resource.


lbSystem.consoleWrite(string)

The console will show "string" plus a line break by Calling this API.


lbSystem.beep(times)

Lobster will beep for several time identified by the parameter.


lbSystem.say(content)

Lobster will call the local API to say the content.


absTime lbSystem.absTime()

Calling this function can return the absolute time right now.


lbSystem.pause()

Calling this function will pause the evaluation of current script.
