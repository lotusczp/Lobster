IMC(IPC3) plugin help

IMC(IPC3) plugin is used to communicate with virtual bit box through serial port using IPC3 protocol.

The plugin read data from virtual bit box every 100ms. Virtual bit box requires continuously writing command otherwise it will consider this point as "comm loss". The plugin will write the value every 200ms.

Settings widget

IMC(IPC3) plugin need to configure all properties of serial port.
