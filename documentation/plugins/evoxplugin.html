Evox plugin help

Evox plugin is used to handle UC800 evox/var protocol.

The plugin send the batch request every 1 second. And send a single write request every 100 ms if has any. If the write command is coming too fast from Lobster main program, at most 100 write commands will be buffered. The rest will be dropped.

Settings widget

Evox plugin need to configure the server ip and port(can not be empty) in Settings widget.


Tool widget

The tool widget shows the network errors.
