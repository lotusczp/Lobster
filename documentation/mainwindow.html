main window help

Main window contains 3 docking widgets: The oscilloscope, the log widget, the watch window.

Those widgets are docking so you can place them anywhere you like. The Windows menu can control the visibility of them.


tool buttons

By Clicking this button, all channels, no matter which transmission it relies on, are started. Once clicked, the start button will change to pause button, it can be used to pause all the transmission. "Pause" means the plugin no longer exchange data with Lobster main program but the hardware resource (serial port, etc.) it occupies are not released.


By Clicking this "Stop" button, the user can completely stop all transmissions and release all occupied resource.
