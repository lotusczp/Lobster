Lobster Help Document

Lobster is a general purpose debug/test tool. It do the transmission work using plugins. All points can be manipulated by Lobster Script. (see Lobster Script for details)

This page gives a typical use case for Lobster. It will satisfy normal usage. Please check the documents of seperate modules for details.

Setup channels

click Tools->Channel Manager to open the channal manager window as follows.

Each channel requires a configuration file for its points and the corresponding transmission plugin.(see plugins for details) For example, we add a channel named "Var" for UC800 evox/var transmission:

Note: The channel name should be unique. But the configuration file and plugin can be used for mutiple channels. For example, we can create a new channal named "Var2" using exact the same "EvoxVar_RTAF.xml" and "EvoxPlugin.dll". The plugin can create mutiple instances if loaded mutiple times.

Select the points

If the plugin and configuration file is loaded correctly, we can open Settings->Select Datato see all the supported points.

You can add/remove points you care by double click the point name, or by the four buttons. In this example, we chose "add all".

All the selected points are added to the watchwindow

Configure the channel if necessary

Click the to see if this channel need any configurations. This configuration is typically related to hardware. For example, the channels using Evox plugin need to configure the server ip and port(can not be empty). The channels using IMC plugin need to configure the serial port stuff such as prot number and baudrate.

Note: Clicking "OK" button here can not close the widget, since other channels still waiting to be configured. It just comfirm the configuration of this single channel. If you need close this widget, clicking the will do.

Start/Stop transmission

By Clicking the , all channels, no matter which transmission it relies on, are started. Each transmission runs in a seperate thread. Once clicked, the start button will change to pause button, it can be used to pause all the transmission. "Pause" means the plugin no longer exchange data with Lobster main program but the hardware resource (serial port, etc.) it occupies are not released. Use "Stop" button to completely stop all transmissions and release all occupied resource.

see main window for details.

Commonly used operations

Double click the "value" cell in the watch window can change the value if this point is writable.

Check/Uncheck in the "Scope" cell can show/unshow the curve of this point in oscilloscope.

Check/Uncheck in the "Log" cell can log/unlog the values of this point into a log file. (see Log system)

For more operations of watchwindow, see Watch window for details.


To change the value of Max Y and Min Y and click "set value" can change Y axis of oscilloscope.

To change the time range can change X axis of oscilloscope.

Note: Time range can only be changed by mouse wheel or UP/DOWN key.

For more operations of oscilloscope, see oscilloscope for details.


Script system

Benefit from Qt framwork, Lobster support the ECMAScript scripting language, as defined in standard ECMA-262. The script is like Javascript and can manipulate the system as well as the points.

By Clicking the can open the script IDE. See script system for script APIs.
